QQ Music API

基于 Cloudflare Pages + D1 的 QQ 音乐 API 服务

搜索

GET /api/search

搜索歌曲、歌手、专辑或歌单

参数 类型 说明
keyword * string 搜索关键词
type string 搜索类型: song, singer, album, playlist (默认 song)
num int 返回数量 (默认 10)
page int 页码 (默认 1)
GET /api/search?keyword=周杰伦&type=song&num=20

歌曲

GET /api/song/url

获取歌曲播放链接

参数 类型 说明
mid * string 歌曲 MID，多个用逗号分隔
quality string 音质: 128, 320, flac (默认 128)
GET /api/song/url?mid=0039MnYb0qxYhV&quality=320
GET /api/song/detail

获取歌曲详细信息

参数 类型 说明
mid string 歌曲 MID (mid 或 id 二选一)
id int 歌曲 ID
GET /api/song/detail?mid=0039MnYb0qxYhV

歌词

GET /api/lyric

获取歌词，支持 LRC、QRC（自动解密）、翻译、罗马音

参数 类型 说明
mid string 歌曲 MID (mid 或 id 二选一)
id int 歌曲 ID
GET /api/lyric?mid=0039MnYb0qxYhV

专辑 / 歌单 / 歌手

GET /api/album

获取专辑详情

参数 类型 说明
mid string 专辑 MID
GET /api/album?mid=002fRO0N4FftzY
GET /api/playlist

获取歌单详情及歌曲列表

参数 类型 说明
id * int 歌单 ID
num int 歌曲数量 (默认 100)
GET /api/playlist?id=8052190267
GET /api/singer

获取歌手信息

参数 类型 说明
mid string 歌手 MID
GET /api/singer?mid=0025NhlN2yWrP4

排行榜

GET /api/top

获取排行榜列表或指定排行榜详情

参数 类型 说明
id int 排行榜 ID，不传则返回所有榜单
num int 歌曲数量 (默认 100)
GET /api/top
GET /api/top?id=4&num=50
